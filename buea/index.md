# AI Tools for Research — University of Buea

A four-session hands-on workshop for graduate students and faculty in science and engineering.

**Dates:** June 29 – July 20, 2026 · Mondays at 4:00 PM WAT (UTC+1)  
**Format:** Interactive online sessions via Google Meet  
**Cost:** Free

**Join:** [meet.google.com/tzf-zjyt-cqh](https://meet.google.com/tzf-zjyt-cqh)  
Dial-in: +1 216-800-8538, PIN 787 282 006#

## Sessions

**Session 1 — June 29**  
Literature review and source analysis: using AI to screen papers, identify patterns, and extract findings.

**Session 2 — July 6**  
Data analysis with AI: exploratory data work, generating insights, maintaining research integrity.

**Session 3 — July 13**  
Academic writing and research methodology: AI-assisted editing, structure refinement, responsible use.

**Session 4 — July 20**  
AI-assisted coding (Python, R, MATLAB) and reference management with Zotero and GitHub.

## Tools

All tools are free and web-based.

| Tool | Purpose |
|------|---------|
| NotebookLM | Interactive research source analysis |
| Claude Projects | AI-assisted research workflows |
| Zotero | Citation and bibliography management |
| GitHub | Version control for research code and data |

## Before You Join

A computer or laptop, internet connection, and a Google account are all you need. Optionally, set up a free [Claude account](https://claude.ai), install [Zotero](https://www.zotero.org), and create a [GitHub account](https://github.com) before Session 1.
