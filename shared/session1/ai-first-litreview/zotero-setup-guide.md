## Install Phase — Desktop App

### 1. Download and install Zotero Desktop

Go to [zotero.org/download](https://www.zotero.org/download/) and download the installer for your operating system. Run it and complete the standard installation.

### 2. Move your data directory to an easy-to-find location

> **⚠️ Critical:** Do **not** put your Zotero data folder inside Google Drive, Dropbox, or any other sync folder. Zotero uses a SQLite database that sync tools will corrupt. Use Zotero's own sync for your library metadata.

By default, Zotero stores your data in a hidden system folder. Move it somewhere you can actually find:

**macOS — recommended path:**
```
/users/[yourname]/documents/zotero
```
Example: `/users/steve/documents/zotero`

**Windows — recommended path:**
```
c:\users\[yourname]\documents\zotero
```
Example: `c:\users\steve\documents\zotero`

**How to change it in Zotero:**

1. Open Zotero
2. Go to **Zotero → Settings** (macOS) or **Edit → Preferences** (Windows)
3. Click the **Advanced** tab
4. Under **Files and Folders**, find **Data Directory Location**
5. Click **Custom** and navigate to your chosen folder
6. Restart Zotero when prompted

## Install Phase — Browser Connector

### 3. Install the Zotero Connector

Go to [zotero.org/download/connectors](https://www.zotero.org/download/connectors) and install the extension for your browser:

- **Chrome / Brave / Edge** → Chrome Web Store install
- **Firefox** → Firefox Add-ons install
- **Safari** → included with Zotero Desktop on macOS

Once installed, you will see a small Zotero icon in your browser toolbar. Its appearance changes depending on what it detects on the current page (book, article, folder, webpage).

<mark>**Zotero Desktop must be open for the connector to save directly to your library.**</mark>

## Step 1 — Create Your First Collection

A Zotero *collection* is a folder. Create one now to hold your workshop materials.

1. In Zotero, right-click **My Library** in the left panel
2. Select **New Collection**
3. Name it something descriptive — e.g., `Power Grid Interconnection` or `AI Tools for Research`

You can create subcollections later. For now, one collection is enough.

---

## Step 2 — Test the Browser Connector (Three Sources)

This is the core skill. Work through all three sources below. Each time, verify that:
- The item appears in your Zotero collection ✓
- A **PDF was downloaded** and attached to the record ✓ (look for the PDF icon under the item)

### 2a. Open-access journal article

1. Navigate to any open-access article — try [PLOS ONE](https://journals.plos.org/plosone/) or search Google for a topic and filter to open access
2. Click the **Zotero Connector icon** in your toolbar
3. Select your collection when prompted
4. Switch to Zotero — the item should appear with a PDF attached

### 2b. Google Scholar

1. Go to [scholar.google.com](https://scholar.google.com)
2. Search for a paper relevant to your research
3. Click through to the publisher page or a PDF link
4. On the article page (not the search results), click the **Connector icon**
5. Confirm the record appears in Zotero

> **Note:** Google Scholar results pages often show a folder icon in the Connector, letting you capture multiple results at once. Try it — select 3–5 papers and save them together.

### 2c. Your university library

1. Log in to your library's discovery system (e.g., SUNY OneSearch, EBSCOhost, Web of Science)
2. Open a full article record
3. Click the **Connector icon**
4. Zotero will usually capture metadata and attempt to download a PDF through your institutional access

> **If no PDF attaches automatically:** Right-click the item in Zotero → **Find Available PDF**. Zotero will query Unpaywall and your library proxy.

## Step 3 — Generate and Import a .bib File from Gemini

This step shows how to seed your library from an AI-generated bibliography — then verify each entry.

### 3a. Generate a .bib file in Gemini

Prompt Gemini (or another LLM) with something like:

```
I am researching [your topic]. Generate a bibliography of 10–15 relevant 
peer-reviewed sources in BibTeX format. Focus on sources from the last 
10 years. Return only the BibTeX entries, no commentary.
```

Copy the output.

### 3b. Save as a .bib file

Paste the output into a plain text editor (TextEdit on Mac set to plain text, Notepad on Windows, or VS Code) and save it as:

```
my-bibliography.bib
```

### 3c. Import into Zotero

1. In Zotero: **File → Import**
2. Select **A file** → navigate to your `.bib` file
3. Choose whether to import into a new collection or an existing one
4. Click **Next / Finish**

The records will appear in your library. **Treat every entry as unverified.** LLMs generate plausible-looking citations that may have incorrect authors, years, titles, or DOIs.

### 3d. Verify each entry

For each imported record:
1. Click the item in Zotero
2. Check the **DOI field** in the right panel
3. Click the DOI link — confirm the paper exists and matches the record
4. If no DOI: Google the title to find the actual source

Delete or correct any records that cannot be verified.

## Step 4 — Find Full Text and Capture in Zotero

For records in your library that are missing PDFs:

### Option A — Automatic (try first)
Right-click the item → **Find Available PDF**
Zotero queries Unpaywall and your library's open-access resolver automatically.

### Option B — Manual capture via browser
1. Find the paper online (publisher site, PubMed, ResearchGate, etc.)
2. On the full-text page, click the **Connector icon** to re-capture with PDF attached
3. If a duplicate is created, merge the records: select both → right-click → **Merge Items**

### Option C — Drag and drop PDF
If you already have the PDF on your computer:
1. Drag the PDF file onto the record in Zotero
2. It attaches as a linked or stored file

## Zotero Sync (Optional but Recommended)

Create a free account at [zotero.org](https://www.zotero.org) and sign in under **Zotero → Settings → Sync**. This backs up your library metadata to Zotero's servers and lets you access your library from any machine.

> Zotero's free account syncs metadata and notes. PDF storage is limited (300 MB free). For unlimited PDFs, use the linked-file architecture with a separate Google Drive or OneDrive folder — but keep your **data directory** off cloud sync (see Install Phase above).
